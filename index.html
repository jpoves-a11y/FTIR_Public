Methodology Info

Depth Selection:

  • Select specific depth from dropdown (100Î¼m, 200Î¼m, etc.)
  • Each depth represents material at different penetration levels
  • Surface layers (100Î¼m) typically show higher oxidation
  • Compare different depths by changing selection and re-running analysis

Deconvolution Process:

  1. Spectral normalization using selected C-H reference band
  2. Baseline correction (linear/polynomial)
  3. Derivative analysis (2nd & 4th order) for peak detection
  4. Gaussian-Lorentzian curve fitting for 6 carbonyl species
  5. Quality assessment (RÂ² > 0.90 target)

Comparative Analysis Methodology
Analysis Process:
  1. Both datasets are processed using identical synovial liquid analysis pipeline
  2. Carbonyl deconvolution is performed for each depth in both datasets
  3. Species concentrations are calculated using extinction coefficients
  4. Differences are computed for matching depths between datasets
Difference Calculations:
  • Absolute: Cpre - Cpost
  • Percentage: ((Cpre - Cpost) / Cpre) x 100%
  • Positive values indicate higher concentration in Pre-Extraction
